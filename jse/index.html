注意事項

  • Zoom、CLEを用いたメディア授業を実施します。グループ番号の半角数字から始まるように、Zoomの名前を設定してください
    例：名前が「高木　優」で担当TAが小南さん（グループ１）の場合は名前を「1 高木　優」に設定。
  • 演習形式の授業のため、出席および課題提出の双方を重視します。課題は全て必ず提出し、合格（全12回分）するようにしてください。特に最終課題の比重が大きくなっています。
    （たとえ授業を欠席しても、課題は提出してください）。
  • 公欠（大学で認められているのは、伝染性疾病（インフルエンザを含む）、教育実習、裁判員制度のみ）は、速やかに届け出てください。
    （※基礎工学部教務係に備え付けの「欠席届（所定書式）」の提出が必要です。疾病等による理由の場合は医師の診断書を添付ください。）
  • 課題は決められたフォーマットにてソースコードを作成し、CLEを通じて担当TA（ティーチングアシスタント）に提出してください。
    課題が出された回の次回の講義開始時間までに提出し、解ければ最高点です。間違えても次々回の講義開始時間までに解ければ加点します。
    力試し課題が解ければボーナス点なので、次々回の講義までに提出してください。
    （文字コードはUTF-8を、改行コードはUnix (LF)を用いてください）

参考

[選択肢1. 自分のPCにC/C++のコンパイラをインストール]

個人のPC（Windowsマシンを想定）でプログラミング演習を行います。
各自以下の環境を整えて、プログラムのコンパイルをできる状態にしてください。
Windows: Cygwinをこちらのチュートリアルを参考にインストールしてください。途中にあるパッケージのインストールを忘れずに行ってください。
テキストエディタとして、Visual Studio CodeやNotepad++など使いやすいものをインストールしてください。
Mac: Terminalにて、Homebrewをインストールし、$ brew install gccとコマンドを入力してgccをインストールしてください。
